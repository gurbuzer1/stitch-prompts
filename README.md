### 1. Morpho — AI Pet Portrait Transformation
> **Prompt:**
> Design a premium dark-themed mobile app for transforming pet photos into artistic portraits. The app should feel luxurious and playful — like a high-end creative studio for pet lovers.
>
> **Screens needed:**
> - **Welcome/Onboarding**: Gorgeous gradient backgrounds with floating pet illustrations, smooth Lottie animations, "Get Started" CTA with haptic feedback feel
> - **Home/Gallery**: Masonry grid of AI-generated pet portraits (oil painting, fantasy warrior, Renaissance noble, space explorer styles). Each card has a subtle parallax effect. Top bar has category filter chips (All, Oil, Fantasy, Space, Renaissance)
> - **Transform Screen**: Large photo preview area, style picker as horizontal scrollable cards with preview thumbnails, "Transform" button with loading shimmer effect
> - **Social Feed**: Instagram-style feed showing community pet portraits. Like/comment/share actions. Each post shows original vs transformed comparison
> - **Profile**: User avatar, stats (portraits created, styles used), gallery of their creations, settings gear icon
>
> **Design language:** Dark mode (#0A0A0F background), gold/purple accent gradients, rounded cards with glassmorphism, smooth 300ms transitions, playful micro-interactions

## Usage Instructions

1. Go to [stitch.withgoogle.com](https://stitch.withgoogle.com)
2. Paste the prompt for the app you want to redesign
3. Let Stitch generate the UI design + code
4. Share the output (screenshots or code) with Clawdy
5. Clawdy integrates it into the actual repo

**Priority order suggested:**
1. Morpho (flagship, most users)
2. Terrain (second flagship)
3. HabitForge, PrayerPal, KiranaBook (popular niches)
4. Rest in batches
